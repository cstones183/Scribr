Implementation Notes
Idle — use as template image
In Xcode, set the idle SVG/PNG as a template image (isTemplate = true). macOS automatically renders it in black on light mode and white on dark mode.
Recording — do not use as template
The coral icon must NOT be a template image — the colour needs to stay coral in both modes. Set isTemplate = false and swap it in when recording starts.
PNG export sizes
Export at 22×22px (1×) and 44×44px (2× Retina). Name them icon.png and icon@2x.png — macOS picks the right one automatically.
Animating the transition
For a smooth swap, use NSStatusItem.button.image and fade between the two using a CABasicAnimation on opacity — keeps it feeling polished.
